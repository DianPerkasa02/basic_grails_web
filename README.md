# README #

## What is this repository for? ##

* Grails basic web 

## How do I get set up? ##

### Environments ###
* Grails version: 3.1.3
    * Groovy version: 2.4.6
* JVM version: 1.8
* Gradle Version: 2.9

### Run Application ###
* Open terminal / cmd
* use command `grails run-app` to run the web app
